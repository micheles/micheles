Module nre

nre - new style regular expressions

The nre module adopt an object-oriented approach to regular expressions. It provides only a class, Regexp, which instances are regular expression objects, or reobj for short. It can be imported with

>>> from nre import Regexp

(notice that from nre import * would have the same effect).

Classes

class Regexp(object):

The class of regular expression objects. Instantiated with the signature

reobj=Regexp(pattern,name=None,doc=None,isgroup=False)

pattern is a (raw) string which can be associated with a valid regular expression; it the pattern contains an sequence ' #', only the chars on the left of ' #' are retained, whereas all the chars on the right of ' #' are assumed to be a comment. If you want to avoid this interpretation, you must escape #. Then the comment is stripped with the pattern and assigned with the docstring, unless an explicit docstring was already provided. if name is not None, associates a name to the reobj; if doc is not None, associates a docstring to the reobj; if isgroup is True, converts the regexp in a named group (?P<name>regexp) or an numbered group (regexp) if name is None.

re-objects are callable. For instance

>>> a=Regexp('x # named group')('a')

returns a named group regular expression, whereas

>>> b=Regexp('x # numbered group')

returns a numbered group regexp.

>>> print a
>>> print b

compose(self,other,oper):

Compose regular expression objects with strings or with themselves. oper is one of the strings "__add__", "__radd__", "__or__", "__ror__".

__ror__(self,other):

Combines a regular expression object to a string or another reobj using | . Works from the right. Example:

>>> print 'x' | Regexp('y')
<reobj '<noname>':x|y>

__radd__(self,other):

Adds a regular expression object to a string or another reobj. Works from the right. Example:

>>> print 'x' + Regexp('y')
<reobj '<noname>':xy>

__or__(self,other):

Combines a regular expression object to a string or to another reobj using | . Works from the left. Example:

>>> print Regexp('x') | 'y'
<reobj '<noname>':x|y>

Works well with named reobj too:

>>> a=Regexp('x',name='a')
>>> b=Regexp('y',name='b')
>>> print a|b
<reobj 'a__b':x|y>

__add__(self,other):

Adds a regular expression object to a string or to another reobj. Works from the left. Example:

>>> print Regexp('x') + 'y'
<reobj '<noname>':xy>

Works well with named reobj too:

>>> a=Regexp('x',name='a')
>>> b=Regexp('y',name='b')
>>> print a+b
<reobj 'a_b':xy>

__call__(self,name=None):

Returns a grouped regular expression object. The group is named if the regexp was already named or if an explicit name is passed.
